🔐 chmod 권한 계산기

755/644 숫자 권한 ↔ -rwxr-xr-x 문자 권한을 즉시 변환합니다. 소유자/그룹/기타 토글과 SUID/SGID/Sticky(4자리)까지 한 번에 확인하세요.

예시:
숫자 권한
755
특수권한 있으면 4자리로 표시 (예: 4755)
문자 권한
-rwxr-xr-x
타입(-/d/l) + 9자리 권한
chmod 명령어
chmod 755 파일명
파일명만 바꿔서 사용

권한 선택

Read(4) · Write(2) · Execute(1)
소유자
값 7
그룹
값 5
기타(others)
값 5
특수 권한 (선택)
특수값 0
SUID/SGID는 실행 위치에 s/S, Sticky는 others 실행 위치에 t/T로 표시됩니다.
복사됨

chmod 권한 계산기 사용법

  • 숫자 권한(755/644)을 입력하면 문자 권한(-rwxr-xr-x)으로 변환됩니다.
  • 문자 권한을 입력하면 숫자로 변환됩니다. (10자리: 타입 포함, 9자리: 타입은 탭 선택 유지)
  • 4자리 권한(예: 4755, 1777)은 SUID/SGID/Sticky를 포함합니다.
