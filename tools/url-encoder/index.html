🔗 URL 인코더/디코더

한글·공백·특수문자가 포함된 URL/쿼리스트링을 안전하게 인코딩(변환)하거나, % 형태 문자열을 사람이 읽을 수 있게 디코딩(복원)합니다.

변환 도구

팁: 쿼리 파라미터 값은 보통 encodeURIComponent가 안전합니다. (Ctrl/⌘ + Enter로 실행)

예시:

URL 인코더/디코더가 필요한 순간

URL에 한글/공백/특수문자가 포함되면 요청이 깨지거나 서버에서 파라미터가 잘못 해석될 수 있습니다. 이 도구로 인코딩/디코딩하여 링크 공유, API 테스트, 로그 해석 등에 활용하세요.

  • 한글 검색어가 포함된 링크가 깨질 때
  • API 테스트에서 쿼리스트링 값을 안전하게 만들고 싶을 때
  • 로그의 %ED%… 형태를 사람이 읽게 복원하고 싶을 때

자주 묻는 질문 (FAQ)

URL에서 한글/공백/특수문자가 깨지지 않도록 %XX 형태로 변환하기 위해 필요합니다.

encodeURIComponent는 파라미터 값 같은 “부분”에 안전하고, encodeURI는 전체 URL에서 예약 문자를 유지합니다.

% 형식이 깨져 있으면 오류가 날 수 있습니다. 입력값을 확인해 주세요.
