⌨️ 한영타 변환기 (확장형)

한/영 전환 실수로 찍힌 문장을 즉시 복구합니다. (입력 즉시 변환 · 숫자/특수문자 유지 · Shift 보정)

자동 감지 숫자·특수문자 유지
ⓘ 자동 감지 기준 보기
  • 입력 문장에서 한글/영문(알파벳) 비율을 비교합니다.
  • 우세한 쪽을 기준으로 변환 방향(영→한 / 한→영)을 자동 결정합니다.
  • 두벌식(표준) 키보드 배열을 매핑하며, 숫자·공백·특수문자는 유지합니다.
  • Shift(대문자) 입력이 섞여 어긋나는 패턴을 보정해 정확도를 개선했습니다.

결과가 반대로 나오면 상단에서 영→한 또는 한→영 모드로 고정해 사용하세요.

변환 모드
감지: 자동
* 두벌식(표준) 기준으로 변환됩니다. 변환 결과가 어색하면 모드를 수동(영→한/한→영)으로 바꿔보세요.

한영타 변환기가 필요한 순간

채팅, 메모, 검색창에서 한/영 전환을 놓친 채로 문장을 입력하면 다시 치기 귀찮습니다. 이 도구는 키보드 입력 실수를 빠르게 복구하기 위해 만들어졌습니다.

  • 한글로 써야 하는데 영어로 찍힌 문장 복구
  • 영어로 써야 하는데 한글로 찍힌 문장 복구
  • 문장 전체 변환 + 숫자·특수문자 유지
  • 입력 즉시 변환 + 복사 버튼
  • Shift(대문자) 보정으로 변환 정확도 개선

자주 묻는 질문 (FAQ)

두벌식(표준) 키보드 배열 기준으로 입력 문장을 키 매핑 방식으로 변환합니다. 자동 모드에서는 입력 내용에서 한글/영문 비율을 비교해 변환 방향을 결정합니다.

숫자·공백·특수문자는 그대로 유지됩니다. 한/영 키에 해당하는 문자만 변환됩니다.

상단에서 영→한 또는 한→영 모드로 고정해 사용하면 됩니다.
