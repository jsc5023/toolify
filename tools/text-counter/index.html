✏️ 글자수 세기 (공백 포함/제외, 단어 수, 바이트 수)

텍스트의 글자 수, 공백 포함/제외, 단어 수, 줄 수, 바이트 수(UTF-8)를 실시간으로 계산해주는 무료 글자수 세기 도구입니다. 자소서, 블로그 글, 공모전, 카카오톡 공지 글자 수 제한을 확인할 때 활용해 보세요.

글자 수 / 통계

공백 포함: 0
공백 제외: 0
단어 수: 0
줄 수: 0
바이트 수: 0

텍스트 입력

아래 입력창에 글을 붙여 넣거나 직접 입력하면 글자 수가 자동으로 계산됩니다.

* 단어 수는 공백(띄어쓰기, 줄바꿈 등)을 기준으로 계산합니다.
* 바이트 수는 UTF-8 인코딩 기준입니다. 한글은 보통 2~3바이트로 계산됩니다.

글자수 세기가 필요한 순간

글자수 세기 도구는 자기소개서, 이력서, 공모전, 블로그 글, 카카오톡 공지처럼 글자 수 제한이 있는 글을 작성할 때 특히 유용합니다.

이 페이지의 글자수 세기 도구는 공백 포함/제외 글자 수는 물론, 단어 수, 줄 수, 바이트 수까지 한 번에 확인할 수 있도록 설계되었습니다. 긴 글을 복사해서 붙여 넣기만 해도 실시간으로 통계를 확인할 수 있습니다.
