URL 파서/쿼리 편집기가 필요한 순간

디버깅/로그 분석/API 테스트 중에는 “URL 파라미터를 빠르게 확인하고 바꿔서 다시 실행”하는 일이 많습니다. 이 도구는 URL을 자동 분해하고, 쿼리 파라미터를 표로 편집해 즉시 URL을 재생성할 수 있도록 구성했습니다.

  • 쿼리 파라미터를 표로 보고 빠르게 수정
  • 중복 key(a=1&a=2)도 그대로 유지
  • 읽기 쉽게 디코딩해서 보고, 생성 시 안전하게 인코딩
  • 여러 줄 URL도 배치 파싱으로 한 번에 확인

입력된 값은 브라우저에서만 처리되며, 서버로 전송하지 않습니다.

자주 묻는 질문 (FAQ)

인코더/디코더는 문자열을 변환하는 도구이고, URL 파서/편집기는 URL을 분해한 뒤 쿼리 파라미터를 표로 수정해 URL을 다시 만드는 편집기입니다.

네. 동일한 key가 여러 번 등장하는 쿼리스트링도 순서대로 표시/편집 가능합니다.

‘값 디코딩해서 표시’를 켜면 표에서 읽기 쉬운 형태로 보여주고, URL 생성 시에는 안전하게 인코딩하여 다시 만들어줍니다.

아니요. 모든 처리는 브라우저에서만 수행됩니다.
