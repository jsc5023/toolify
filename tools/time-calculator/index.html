⏰ 시간 계산기 (시간 차이 / N시간 후·전 / 근무시간)

두 시간(날짜+시간) 사이 차이 계산, N시간/분/초 후·전 계산, 근무시간(휴게 제외) 계산, 초·분·시간 변환까지 한 번에 할 수 있는 무료 시간 계산기입니다.

두 시간 사이 차이

시작/종료를 날짜+시간으로 입력하면 자정을 넘어가는 경우도 정확하게 계산됩니다.

N시간/분/초 더하기 · 빼기

근무시간 계산 (휴게 제외)

근무 시작~종료의 총 시간에서 휴게시간(분)을 제외해 실 근무시간을 계산합니다.

초 · 분 · 시간 변환

시간 계산기가 필요한 순간

시간 계산기는 출퇴근 기록, 근무 시간 산정, 일정 사이 공백 확인, 프로젝트 작업 시간 측정처럼 일상과 업무에서 자주 쓰입니다. 날짜가 넘어가거나 휴게시간을 제외해야 하는 경우, 머리로 계산하면 실수하기 쉬워 계산기를 활용하면 편합니다.

  • 두 일정 사이의 시간 차이를 정확히 알고 싶을 때
  • 지금부터 N시간/분/초 후(전)가 언제인지 확인할 때
  • 근무시간에서 점심/휴게를 제외한 실 근무시간이 필요할 때
  • 초·분·시간 단위를 빠르게 변환해야 할 때

이 페이지의 시간 계산기는 브라우저에서만 동작하며 입력 값은 저장되지 않습니다.
