# 원본 감지 단위 KST 상대시간

Unix Timestamp 변환기가 필요한 순간

서버 로그, 이벤트 추적, API 응답 등에서 Unix Timestamp(초/밀리초)를 자주 확인하게 됩니다. 이 도구는 초/밀리초 자동 감지, KST(한국시간) 기본 출력, UTC/ISO 동시 제공, 상대시간 표시로 “지금 기준으로 언제인지”를 빠르게 파악할 수 있도록 구성했습니다.

  • 로그의 timestamp를 KST/UTC로 빠르게 확인
  • API 테스트 시 날짜/시간을 timestamp로 변환
  • 여러 줄 로그를 배치로 붙여넣고 한 번에 변환
  • 현재 기준 상대시간(몇 분 전/후) 확인

입력된 값은 브라우저에서만 처리되며, 서버로 전송하지 않습니다.

자주 묻는 질문 (FAQ)

보통 10자리면 초(s), 13자리면 밀리초(ms)입니다. 이 도구는 입력 길이를 기반으로 자동 감지하며, 필요하면 단위를 직접 선택할 수 있습니다.

네. 결과에서 KST(한국시간), UTC, ISO 8601 값을 동시에 제공합니다.

배치 변환 탭에서 여러 줄을 붙여넣으면 각 줄에서 timestamp 후보를 자동 추출하여 표로 변환합니다.

현재 시각(Date.now)과 비교하여 “방금 전/몇 분 전/몇 시간 후”처럼 표시합니다.
